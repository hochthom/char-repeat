# char-repeat
Word Art Generator

Example:
--------
Create a mask first:
```
%run create_mask.py NaN Source_Code_Pro/Source_Code_Pro-Black.ttf
```
Create the image with:
```
%run create_image.py -s 16 mask_NaN.png Source_Code_Pro/Source_Code_Pro-Black.ttf
```
